 

# Customer Analysis

A Data Analytics project that explores customer behavior using **Python, SQL, and Power BI**.

## Overview

This project analyzes customer behavior using Python, SQL, and Power BI. The dataset is cleaned and explored using Python, analyzed with SQL, and visualized through an interactive Power BI dashboard. A report and presentation were also created to summarize the key findings.

---

## Dataset

* **Dataset:** Customer Behavior Dataset
* **Format:** CSV
* **Contents:** Customer demographics, purchase history, subscription status, payment methods, shipping details, and product information.

---

## Tools Used

* Python (Pandas)
* PostgreSQL / MySQL / SQL Server
* Power BI
* Gamma

---

## Project Steps

1. Load the dataset using Python.
2. Perform Exploratory Data Analysis (EDA).
3. Clean and prepare the data.
4. Analyze the data using SQL queries.
5. Build an interactive Power BI dashboard.
6. Create a report and presentation.

---

## Dashboard

The dashboard includes:

* Sales Overview
* Customer Insights
* Product Performance
* Subscription Analysis
* Payment Method Analysis

---

## Results

* Identified customer purchasing patterns.
* Analyzed customer preferences.
* Generated business insights using SQL and Power BI.
* Presented the findings through reports and dashboards.

---

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Run the Python notebook for data cleaning and EDA.
4. Execute the SQL queries.
5. Open the Power BI dashboard (`.pbix` file).
